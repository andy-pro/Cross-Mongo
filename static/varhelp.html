Помощь

Для изменения свойств плинта нужно ввести его имя (отображается символом "~" и розовым цветом).

Для добавления плинта нужно ввести новое имя (отображается символом "+" и желтым цветом).

Нумерация пар плинтов устанавливается только для добавляемых плинтов, если не установлен "Применить для всех".

В групповых операциях можно использовать переменные, указанные в таблице.


Переменные

В названиях плинтов
%числосчетчик плинтов
В названиях пар
%Aтекущие
%Eиз редактора. Каждый плинт с новой строки, разделитель между парами - Tab (формат Excel)
%числосчетчик плинтов
%Pчислосчетчик пар
%Dчислосквозной счетчик пар
В общих сведениях
%Aтекущие
%Cкросс (на другом конце кабеля)
%Vвертикаль ...
%Mплинт ...
%числосчетчик плинтов
